AutoMCP — MCP tools optimization

Explore UX for optimizing MCP tool metadata: better descriptions and input schema so agents select and call tools more reliably. Connect to an MCP server, review tools, run optimization, and compare before/after.

Flow

1. Connect MCPServer URL, auth; discover tools 2. Tool listAll tools; name, description; optional health 3. Tool metadataView/edit name, description, inputSchema 4. Run optimizationOptimize descriptions and/or schema 5. Before / afterCompare original vs optimized metadata

What AutoMCP optimizes

  • Tool descriptions — Clearer, agent-friendly text so the LLM picks the right tool.
  • Input schema — Parameter names, types, and per-parameter descriptions (JSON Schema) tuned for LLM consumption.
  • Optional: prompt layer around tool listing (how tools are presented in the system message).
