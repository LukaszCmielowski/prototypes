fetch_weather — Description

VersionText
Original Gets the weather for a location.
Optimized Returns current weather conditions for a given location. Use when the user asks about weather, temperature, or conditions in a city or region. Accepts location (string) and optional unit (celsius|fahrenheit).

fetch_weather — Input schema (excerpt)

VersionPropertyDescription
Original location Location
Optimized location City name or region, e.g. 'San Francisco' or 'London'. Required.
