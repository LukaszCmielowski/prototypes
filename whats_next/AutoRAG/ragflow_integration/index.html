AutoRAG + RAGFlow / LightRAG / OpenRAG

Use AutoRAG (document loading, chunking, RAG template optimization) with popular open-source RAG engines. Connect a backend, run optimization, then sync or deploy the best config into RAGFlow, LightRAG, or OpenRAG for production.

Supported backends

  • RAGFlow (infiniflow/ragflow) — RAG + Agent context layer; sync optimized chunks & prompts to a RAGFlow project.
  • LightRAG (HKUDS/LightRAG) — Simple, fast RAG; export optimized parsing/chunk config for LightRAG ingestion.
  • OpenRAG (langflow-ai/openrag) — Langflow + Docling + Opensearch; deploy optimized pipeline as OpenRAG-compatible config.
