Pipeline → backend mapping

AutoRAG runs document loading, text extraction, chunking, and RAG template optimization. This view shows how outputs map to your chosen backend (RAGFlow).

AutoRAG pipeline → RAGFlow

document-loader → text-extraction → search-space-prep → rag-templates-optimization → RAGFlow

Optimized chunks and retrieval/generation templates are synced to RAGFlow as dataset + prompt config. Test data is used for evaluation in both AutoRAG and (optionally) RAGFlow.
