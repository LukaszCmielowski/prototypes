# Legacy RHOAI AutoRAG evaluation

Adds LLM-as-a-Judge to pre-exported RHOAI evaluation results (separate from the ai4rag GAM notebooks).

## Layout

```
rhoai/
  rhoai_autorag_evaluation.ipynb
  evals/                          # input: RHOAI per-question JSON exports
    evaluation_results_5.txt
    evaluation_results_11.txt
  outputs/                        # generated by the notebook
    rhoai_autorag_with_llmaj.csv
    rhoai_pattern_*_with_llmaj.csv
```

## Run

```bash
cd AutoRAG/comparative_analysis/autorag/rhoai
jupyter notebook rhoai_autorag_evaluation.ipynb
```

Requires OGX credentials in `../../lightrag/POC/.env`.

## Add patterns

Drop new `evaluation_results*.txt` files into `evals/` and re-run the notebook.
